## 🚀 Featured Portfolio Project: Ezi Bank Fraud Detection & BI Dashboard

**Overview**
Developed an end-to-end machine learning pipeline and interactive business intelligence dashboard to detect fraudulent banking transactions. Handled a massive 1-million-row dataset with a severe class imbalance (1.15% fraud rate) to deliver a secure, cost-sensitive AI solution.

**Key Skills Demonstrated**
* **Data Science:** Python, Pandas, Exploratory Data Analysis (EDA).
* **Machine Learning:** XGBoost, SMOTE (Class Balancing), Cost-Sensitive Learning.
* **Feature Engineering:** Created custom business-logic variables (Velocity Ratios, High-Risk Device Flags).
* **AI Explainability:** SHAP (Beeswarm and Bar plots) to decode algorithmic decision-making.
* **Business Intelligence:** Power BI (Interactive visualizations, DAX, single-page scrollable dashboards).

**The Challenge**
Ezi Bank faced a critical 1.15% fraud rate. The business goal was to maximize the detection of bad actors without relying on a rigid, black-box algorithm that couldn't be explained to stakeholders.

**The Solution & Results**
1.  **Optimized AI:** Trained an XGBoost model and adjusted the decision threshold from 0.5 to 0.1 to prioritize the bank's security.
2.  **High-Impact Metrics:** Achieved an impressive **92.9% Recall** (Fraud Detection Rate) and an 87.2% ROC-AUC score.
3.  **Actionable Tooling:** Exported the 200,000-record test set into a custom, four-part Power BI dashboard (Executive Overview, Fraud Drivers, Fraud Investigation, and Model Performance) to allow investigation teams to act on the AI's predictions instantly.

**Project Conclusion**
We successfully built a powerful AI model that catches nearly all scammers, achieving an impressive 92.9% fraud detection rate. We accomplished this by balancing the dataset with SMOTE and lowering the model's threshold to prioritize the bank's security. However, we missed out on having highly accurate alerts, as our precision dropped to just 2.2%. This happened because aggressively catching more fraud naturally triggers more false alarms for innocent customers, which was a necessary business trade-off to stop financial losses.
